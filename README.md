## 🛠 Installation & Set Up

1. 패키지 설치

   ```sh
   npm i
   ```

2. 로컬 실행

   ```sh
   npm run dev
   ```
   
3. "sharp" module error가 발생할 때

   ```sh
   npm install --platform=linux --arch=x64 sharp
   ```
